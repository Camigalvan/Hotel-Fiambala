╔════════════════════════════════════════════════════════════════╗
║               🌐 PASO 5: PUBLICAR EN INTERNET                 ║
╚════════════════════════════════════════════════════════════════╝

  OPCIÓN 1: Netlify (Recomendado)
  ─────────────────────────────────
  1. Ve a netlify.com
  2. Haz clic en "Add new site"
  3. Arrastra la carpeta "Hotel fiambala"
  4. ¡Tu sitio está online en segundos!

  OPCIÓN 2: GitHub Pages (Gratis)
  ─────────────────────────────────
  1. Crea cuenta en github.com
  2. Nuevo repositorio "hotel-fiambala"
  3. Sube los archivos
  4. En Settings → Pages → Deploy from branch
  5. URL: usuario.github.io/hotel-fiambala

  OPCIÓN 3: Hosting tradicional (000webhost)
  ──────────────────────────────────────────
  1. Crea cuenta gratis
  2. Descarga FileZilla (FTP)
  3. Sube archivos via FTP
  4. Tu sitio estará online

  OPCIÓN 4: Dominio personalizado
  ───────────────────────────────
  1. Compra dominio (GoDaddy, Namecheap)
  2. Apunta al hosting (DNS)
  3. ¡Hotel Fiambalá en la web! 🎉

╔════════════════════════════════════════════════════════════════╗
║                    🆘 NECESITAS AYUDA?                        ║
╚════════════════════════════════════════════════════════════════╝

  Problema: El sitio no abre
  Solución: 
  → Verifica que estés abriendo index.html
  → Intenta con navegador diferente (Chrome, Firefox)
  → Usa Live Server en VS Code

  Problema: Los enlaces no funcionan
  Solución:
  → Verifica que los archivos HTML estén en la misma carpeta
  → Usa rutas relativas (sin http://)
  → Ejemplo: href="nosotros.html"

  Problema: Imágenes no cargan
  Solución:
  → Coloca imágenes en carpeta "images/"
  → Usa ruta: images/nombre-foto.jpg
  → Revisa que el nombre sea exacto

  Problema: WhatsApp no abre
  Solución:
  → Verifica que uses formato: https://wa.me/54xxxxx
  → Sin símbolos: +54 → 54
  → Número completo sin espacios

  Problema: Quiero cambiar el diseño
  Solución:
  → Modifica css/style.css
  → Cambia colores en :root
  → Ajusta tamaños de fuente
  → NO requiere conocimiento avanzado
